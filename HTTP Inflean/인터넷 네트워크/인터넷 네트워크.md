### TCP 3 way handshake
![[Pasted image 20240314003422.png]]

1. 클라이언트: `SYN`을 보냄
2. 서버: `SYN + ACK`로 응답보냄
3. 클라이언트: `ACK` (최근은 최적화가 되어서 3번 동시에 데이터를 보냄)

-> 연결이 되었음을 확인되고 데이터 전송이 가능!

*논리적으로만 연결이 된 것이다*
