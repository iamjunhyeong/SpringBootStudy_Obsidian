### index 페이지
`Application Root: /` 요청 시 보여주는 페이지

**특징**
- 기본 리소스 위치 내 `index.html`을 찾아보고 있으면 제공
- `index.템플릿` 찾아보고 있으면 제공
- 둘 다 없으면 스프링 부트에서 제공하는 White Label Error Page
